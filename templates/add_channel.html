¿Cómo encontrar el ID del canal?

1

Método 1: Desde la URL del canal

Ve al canal en YouTube. Si la URL es como:

youtube.com/channel/UCXXXXXXXXXXXXXXXXXXXXXX

El ID es la parte después de "/channel/"

2

Método 2: Desde el código fuente

En la página del canal, haz clic derecho y selecciona "Ver código fuente"

Busca (Ctrl+F) por: "channelId"

Encontrarás algo como:

"channelId":"UCXXXXXXXXXXXXXXXXXXXXXX"
3

Método 3: Usando herramientas online

Puedes usar sitios web como:

  • commentpicker.com/youtube-channel-id.php
  • youtube.com/account_advanced
